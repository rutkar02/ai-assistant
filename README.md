# 🤖 AI Assistant From Scratch

> Building a production-style modular AI assistant from first principles while learning AI Engineering.

This repository documents my journey of understanding and building an AI assistant instead of simply following tutorials.

The goal is not just to create a chatbot, but to understand how modern AI systems are engineered.

---

# 🎯 Objectives

- Learn AI Engineering by building
- Understand every component deeply
- Build reusable modules instead of one-off scripts
- Keep a complete development history through Git commits

---
